# Spares store

Test website. Three page site for auto parts store.
>The layout was purchased from HTMLacademy

## Objective 

1. Learn how to work with:
  - Flexbox
  - SASS
  - Prepros
  - Js (jQuery)
  - Media queries
  - CSS-transform
  
   
2. Implement a fixed burger menu



## Stack 

- HTML5
- CSS3
- JS
- Methodology BEM
